---
Toast Documentation
---

**BToast** is a custom function, designed to display a toast with a message

To use **BToast**, call it as a function and provide an object with following properties

>- **title** (optional): A string representing toast title
>- **message**: A string containing the toast message
>- **type** (optional): A string specifying the toast type. The type can be the following options: info, success, warning or danger. Default is danger
>- **top** (optional): A boolean determining if the toast will be in top of the page
>- **bottom** (optional): A boolean determining if the toast will be in bottom of the page
>- **right** (optional): A boolean determining if the toast will be in right of the page
>- **left** (optional): A boolean determining if the toast will be in left of the page
>- **timeout** (optional): A number specifying after which time the toast will dispear. The mesuare is milliseconds
>- **button** (optional): A string specifying the button text
>- **action** (optional): A function that will be executed when the button is clicked


```js
this.$toast({
    title: 'Title',
    message: 'message',
    type: 'info',
    top: true,
    right: true,
    timeout: 3500,
})

// or

import { inject } from 'vue'
const toast = inject('toast')
toast({
    title: 'Title',
    message: 'message',
    type: 'info',
    top: true,
    right: true,
    timeout: 3500,
})
```

Remember to provide just one attribute for which (x, y) direction. For example: (bottom and left), (top and right), etc.

To make **BToast** work, add the following code in your app.vue

```html
<b-toast />

<!-- or -->

<b-toast></b-toast>
```
