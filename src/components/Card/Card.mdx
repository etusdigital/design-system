# Name: Card
## Component Overview

**Purpose**: A simple container React component that provides a card-like appearance with background, border, shadow, and rounded corners for grouping related content.

**Import**: Automatic - no need to import any DS components

### Basic Usage

```tsx
<Card className="w-fit">
    <div className="w-fit flex flex-col p-base gap-base">
        <h1 className="font-bold text-lg m-none">Form</h1>
        <input label="Name" />
        <button>Save</button>
    </div>
</Card>
```

### Props API

#### className
Optional CSS classes to apply additional styling to the card. Type: `string` (optional)

```tsx
<Card className="max-w-sm">...</Card>
```

---

### Events API

Card does not emit any custom events. It's a simple container React component that passes through standard DOM events from the underlying div element.

---

### Children API

#### children
The main content for the card. Accepts any content that should be displayed inside the card container.

```tsx
<Card>
    children: default
</Card>
```

**Important Notes:**
- The children content determines the card's dimensions - the card adapts to its content
- Consider adding padding classes to the content for proper spacing
- The card provides the visual container, but content layout is handled by what you put as children
- Use semantic HTML elements within the card for better accessibility
