  <action>
    IMPORTANT: Only fix type errors. Do NOT change component behavior, DOM output, or prop interfaces. All changes must be type-level only (casts, type guards, unused variable removal, type annotations).

    Run `npx tsc -b tsconfig.lib.json --pretty 2>&1` first to get the exact current error list. Then fix each error category:

    **Category 1: `child.props` is of type `unknown` (Input, TagInput, Tooltip, FileUpload)**
    Add `React.isValidElement(child)` type guard before accessing `child.props`, then cast props:
    ```typescript
    if (React.isValidElement(child)) {
      const childProps = child.props as Record<string, unknown>;
      // use childProps instead of child.props
    }
    ```

    **Category 2: `Property 'style' does not exist on type IconProps` (ColorPicker, Profile)**
    Add `style?: React.CSSProperties` to the Icon component's props interface in `src/components/Icon/Icon.tsx`.

    **Category 3: `Property 'size' does not exist on type IconProps` (Profile)**
    Add `size?: string` to IconProps in `src/components/Icon/Icon.tsx`. If Profile uses `<Icon size="...">`, this makes it type-safe.

    **Category 4: `Property 'placeholder' does not exist` on div (RichTextEditor)**
    Replace `placeholder` attribute on contentEditable div with `data-placeholder` attribute. Keep the same value. The existing CSS should already use `[data-placeholder]` or add a CSS rule if needed:
    ```css
    [data-placeholder]:empty::before {
      content: attr(data-placeholder);
      color: var(--text-placeholder, #9ca3af);
    }
    ```

    **Category 5: `findLastIndex` does not exist (Pagination)**
    Update `tsconfig.lib.json` `lib` field from `["ES2020", "DOM", "DOM.Iterable"]` to `["ES2023", "DOM", "DOM.Iterable"]`. This adds `Array.prototype.findLastIndex` support.

    **Category 6: Unused variables/imports**
    - `'e' declared but never read` (Sidebar.tsx): prefix with `_e` or remove if in catch block
    - `'parent' declared but never read` (Tree.tsx): prefix with `_parent` or remove
    - `'useRef' declared but never read` (Colors.tsx): remove unused import
    - `'Label' declared but never read` (FileUpload.tsx): remove unused import
    - `'tooltipMinWidth' declared but never read` (MetricCard.tsx): remove unused variable
    - `'useClickOutside' declared but never read` (Container.tsx): remove unused import

    **Category 7: `Type 'Function' not assignable to MouseEventHandler` (Icon.tsx)**
    Change `onClick` prop type from `Function` to `React.MouseEventHandler<HTMLSpanElement>`.

    **Category 8: `Expected 1 arguments, but got 0` (TagInput)**
    Find the function call with missing argument and add the required argument.

    **Category 9: `Type 'unknown' not assignable to ReactNode` (Profile)**
    Add explicit type assertion: `(children as React.ReactNode)` or add a type guard.

    After all fixes, run `npx tsc -b tsconfig.lib.json --pretty 2>&1` again to confirm zero errors.
