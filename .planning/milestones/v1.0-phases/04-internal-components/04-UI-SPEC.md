## Interaction Contracts

Since Phase 4 components are structural utilities, interaction patterns are the primary design contract:

### Overlay
- **Backdrop:** Fixed full-screen, `bg-negative` at 60% opacity, z-index configurable (default 1000)
- **Fade transition:** 0.5s ease opacity (0 → 0.6 enter, 0.6 → 0 leave) via `useTransition` hook
- **Click dismiss:** Click on backdrop fires `onClick` — consumer decides whether to close
- **Portal:** Renders into `document.body` via `createPortal`

### Container
- **Toggle:** Click or Space key toggles expanded state (disabled prevents toggle)
- **Arrow rotation:** `keyboard_arrow_down` icon rotates 180deg on expand (300ms transition)
- **Expand animation:** Content slides via translate-y from -10px to 0 (300ms) with max-height transition
- **Click-outside dismiss:** `closeOnBlur` prop (default true) — uses `useClickOutside` hook with `mousedown` event
- **Label auto-width:** Content min-width syncs to container scroll-width via MutationObserver
- **Focus outline:** `outline-primary-default` on focus, `outline-neutral-default` in rest state

### ExpandableContainer
- **Positioning:** Absolute (`z-[80]`) when expanded, relative when collapsed
- **Alignment:** `alignRight` prop toggles `left-0` vs `right-0`
- **Card styling:** `shadow-neutral-selected`, `border-xxs border-neutral-default`, `rounded-sm`

### SelectContainer
- **Resize tracking:** ResizeObserver + MutationObserver on container, 200ms initial resize delay
- **Content max-height:** `max-h-[12em]` when `dontHaveMaxHeight` is false — scrollable overflow
- **Slide animation:** Content translates from -100% to 0 on expand (100ms ease-out)
- **Actions footer:** Bottom section with `border-t-xxs border-neutral-default`, `p-xs` padding

### SelectContent
- **Search auto-focus:** Input receives focus on expand via `useEffect` with `expanded` dependency
- **Icon states:** Default neutral → expanded primary → disabled neutral-disabled → error danger
- **Search input:** No border, no shadow, no outline — inherits container background

### Group
- **Layout:** `inline-flex`, `flex-col` when vertical, `items-end` when horizontal
- **Context:** Provides `value`, `select(value)`, `disabled` via `GroupContext`

### Option
- **States:** Default (neutral text) → hover (primary text + primary surface bg) → selected (primary-selected text + primary surface bg) → disabled (pointer-events-none, disabled text)
- **ARIA:** `role="option"`, `tabindex="0"`
- **Padding:** Inherits `p3` typography, `gap-xs` for icon spacing
