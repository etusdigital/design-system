  <action>
Read `Tooltip.vue` to extract the full positioning logic, props, and CSS.

Create `Tooltip.tsx` following the RESEARCH.md code example closely:
1. **Props interface:** `labelValue?: string`, `position?: 'top' | 'bottom' | 'left' | 'right'`, `children?: React.ReactNode`, `className?: string`. Plus any additional props from Vue source (check for `style` passthrough).
2. **Named slot "label":** Create `Tooltip.Label` compound sub-component per locked decision: `Tooltip.Label = function TooltipLabel({ children }: { children?: React.ReactNode }) { return <>{children}</>; }`
3. **Hover state:** `useState(false)` for `isHovering` + `onMouseEnter`/`onMouseLeave`.
4. **Transition:** `useTransition(isHovering, { duration: 500 })` returns `{ isMounted, isActive }`.
5. **Portal rendering:** `{isMounted && createPortal(<tooltip-div>, document.body)}`.
6. **Position calculation:** `useEffect` triggered by `[isMounted, position]`:
   - Get trigger element rect via `contentRef.current.firstElementChild || contentRef.current`.
   - Calculate tooltip position using `getBoundingClientRect()` + tooltip `offsetWidth`/`offsetHeight`.
   - Set `style.left` and `style.top` on `tooltipRef.current`.
   - Use `computedStyleMap().get('--spacing-xxs')` with fallback to `4` for padding (same as Vue).
7. **Scroll close:** In the same `useEffect`, add `document.addEventListener('wheel', closeHandler)` with cleanup `return () => document.removeEventListener('wheel', closeHandler)`.
8. **Wrap `calculatePosition` in `useCallback` with `[position]` dependency** to avoid stale closure (Pitfall 4).

Create `Tooltip.module.css`:
- `.tooltip`: `z-index: 1004`, `position: fixed`, `display: flex`, `align-items: center`, `opacity: 0`, `transition: opacity 500ms ease-out`.
- `.active`: `opacity: 1`.
- `.tooltipTriangle`: Arrow/caret styles from Vue source.
- `.tooltipContent`: Content wrapper styles from Vue source.
- Position-specific classes (`.top`, `.bottom`, `.left`, `.right`) for triangle/arrow direction.
- Extract ALL styles from Vue `<style scoped>` — do not invent new styles.
