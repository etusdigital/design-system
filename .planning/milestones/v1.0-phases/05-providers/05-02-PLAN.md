**Types:**

```typescript
type ToastType = 'info' | 'success' | 'warning' | 'danger' | 'neutral';

interface ToastOptions {
  id?: string;
  title?: string;
  message?: string;
  type?: ToastType;
  top?: boolean;
  bottom?: boolean;
  left?: boolean;
  right?: boolean;
  timeout?: number | null;
  buttonLabel?: string;
  action?: () => void;
}

interface ToastItem extends Required<Omit<ToastOptions, 'timeout' | 'action'>> {
  visible: boolean;
  timeout: number | null;
  action: (() => void) | undefined;
}

type ToastAction =
  | { type: 'ADD'; toast: ToastItem }
  | { type: 'HIDE'; id: string }
  | { type: 'REMOVE'; id: string };

interface ToastContextValue {
  toast: (options: ToastOptions) => { id: string; close: () => void };
}
```

**Reducer:**

```typescript
function toastReducer(state: ToastItem[], action: ToastAction): ToastItem[] {
  switch (action.type) {
    case 'ADD':    return [...state, action.toast];
    case 'HIDE':   return state.map(t => t.id === action.id ? { ...t, visible: false } : t);
    case 'REMOVE': return state.filter(t => t.id !== action.id);
  }
}
```

**ToastProvider:**
- `createContext<ToastContextValue | null>(null)` for `ToastContext`
- `useReducer(toastReducer, [])` for toast list
- `useRef<Map<string, ReturnType<typeof setTimeout>>>` for `timerMapRef` (auto-dismiss timers)
- `useEffect` cleanup on unmount: clear all timers in `timerMapRef`

**toast(options) function:**
- Generate ID: `const id = options.id ?? crypto.randomUUID()`
- Default toast values: `title: ''`, `message: ''`, `type: 'danger'` (Vue default), `top: false`, `bottom: false`, `right: false`, `left: false`, `buttonLabel: ''`, `action: undefined`
- Spread options over defaults, set `visible: true`
- `dispatch({ type: 'ADD', toast })`
- Auto-dismiss: `const timeout = options.timeout !== undefined ? options.timeout : 5000`. If `timeout` is truthy (not 0, not null), set `setTimeout(() => removeToast(id), timeout)` and store in `timerMapRef`
- Return `{ id, close: () => removeToast(id) }`

**removeToast(id) function:**
- Clear timer from `timerMapRef` if exists
- `dispatch({ type: 'HIDE', id })`
- `setTimeout(() => dispatch({ type: 'REMOVE', id }), 600)` (600ms matches slide animation)

**Render:**
- `<ToastContext.Provider value={{ toast: addToast }}>` wrapping `{children}` and the portal content
- Portal: `createPortal(<ToastContainers toasts={toasts} onClose={removeToast} />, document.body)` (guard with `typeof document !== 'undefined'`)

**ToastContainers component (internal):**
- 4 containers: `[{ vertical: 'top', horizontal: 'left' }, { vertical: 'top', horizontal: 'right' }, { vertical: 'bottom', horizontal: 'left' }, { vertical: 'bottom', horizontal: 'right' }]`
- Each container: `<div className={clsx('toast-container', container.vertical, container.horizontal)}>` with filtered toasts where `toast[container.vertical] && toast[container.horizontal]`

**ToastItemComponent (internal):**
- `useTransition(toast.visible, { duration: 600 })` for slide animation
- Slide class: `toast.right ? 'slide-right' : 'slide-left'`
- When `isMounted`: render `<div className={clsx('toast-wrapper', slideClass, isActive && 'active')}>` wrapping `<Alert title={toast.title} message={toast.message} type={toast.type} iconPosition="center" closable onClose={onClose} actions={toast.buttonLabel ? <Button size="small" variant="secondary" color={toast.type} className="whitespace-nowrap" onClick={toast.action}>{toast.buttonLabel}</Button> : undefined} className="toast" />`

**useToast hook:**
```typescript
export function useToast(): ToastContextValue {
  const ctx = useContext(ToastContext);
  if (!ctx) throw new Error('useToast must be used inside ToastProvider');
  return ctx;
}
```
