**ConfirmProvider implementation:**
- `createContext<ConfirmContextValue | null>(null)` for `ConfirmContext`
- `useState<ConfirmState | null>(null)` for pending confirm dialog state
- `useRef<((v: boolean) => void) | null>(null)` for `resolverRef` — stores Promise resolve callback

**confirm(options) function:**
```typescript
function confirm(options: ConfirmOptions): Promise<boolean> {
  return new Promise<boolean>((resolve) => {
    resolverRef.current = resolve;
    setState({
      title: options.title ?? '',
      message: options.message ?? '',
      acceptLabel: options.acceptLabel ?? 'Confirm',
      cancelLabel: options.cancelLabel ?? 'Cancel',
    });
  });
}
```

**accept() function:**
- `resolverRef.current?.(true)`
- `resolverRef.current = null`
- `setState(null)`

**cancel() function:**
- `resolverRef.current?.(false)`
- `resolverRef.current = null`
- `setState(null)`

**Render:**
```tsx
<ConfirmContext.Provider value={{ confirm }}>
  {children}
  <Dialog value={state !== null} noOutsideClose className="confirm">
    {state && (
      <div className="flex flex-col p-xl gap-sm">
        {state.title && (
          <h2 className="font-bold text-lg text-neutral-foreground-high">{state.title}</h2>
        )}
        {state.message && (
          <p className="text-sm text-neutral-foreground-low">{state.message}</p>
        )}
        <div className="flex justify-end w-full gap-xs mt-sm">
          <Button variant="plain" onClick={cancel}>{state.cancelLabel}</Button>
          <Button onClick={accept}>{state.acceptLabel}</Button>
        </div>
      </div>
    )}
  </Dialog>
</ConfirmContext.Provider>
```

Notes:
- Dialog `value={state !== null}` — dialog is open when there is a pending confirm state
- Dialog does NOT need `onChange` handler — closing is handled only by accept/cancel buttons. The `noOutsideClose` prop prevents backdrop close.
- Tailwind classes match Vue Confirm.vue template exactly: `p-xl gap-sm`, `text-lg text-neutral-foreground-high`, `text-sm text-neutral-foreground-low`, `gap-xs mt-sm`
- Cancel button: `variant="plain"` (no background, text only)
- Accept button: default variant (filled primary green)

**useConfirm hook:**
```typescript
export function useConfirm(): ConfirmContextValue {
  const ctx = useContext(ConfirmContext);
  if (!ctx) throw new Error('useConfirm must be used inside ConfirmProvider');
  return ctx;
}
```
