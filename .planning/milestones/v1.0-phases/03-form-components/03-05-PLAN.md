**PINInput.tsx** — Replace stub with `React.forwardRef` + `useImperativeHandle`:

Types:
```typescript
export interface PINInputHandle {
  focus: () => void;
  clear: () => void;
}

interface PINInputProps {
  value?: string;
  defaultValue?: string;
  onChange?: (value: string) => void;
  onComplete?: (value: string) => void;
  length?: number;
  disabled?: boolean;
  placeholder?: string;
  separator?: number;     // show separator line after every N fields
  type?: 'text' | 'password';
  mask?: string;
  otp?: boolean;          // sets autocomplete="one-time-code"
  className?: string;
}
```

Component: `export const PINInput = React.forwardRef<PINInputHandle, PINInputProps>(function PINInput(props, ref) { ... })`

Internal state:
- `const [values, setValues] = useState<string[]>(() => { if (value) return value.split('').slice(0, length); return Array(length ?? 6).fill(''); })`
- Sync from controlled `value` prop: `useEffect` watching `value` to update internal values array
- `const inputRefs = useRef<(HTMLInputElement | null)[]>([])`
- Computed: `const joinedValue = values.join('')`
- Fire `onChange(joinedValue)` when values change
- Fire `onComplete(joinedValue)` when all fields are filled (every value non-empty)

`useImperativeHandle(ref, () => ({ focus: () => { const firstEmpty = values.findIndex(v => !v); inputRefs.current[firstEmpty !== -1 ? firstEmpty : 0]?.focus(); }, clear: () => { setValues(Array(len).fill('')); inputRefs.current[0]?.focus(); } }))`

Key handlers (per field at index `i`):
- **onInput**: Set value at index `i` (new array!), if value present auto-advance focus to `inputRefs.current[i+1]?.focus()`
- **onKeyDown Backspace**: If field empty, focus previous field `inputRefs.current[i-1]` and select its content
- **onKeyDown ArrowLeft**: focus `inputRefs.current[i-1]`
- **onKeyDown ArrowRight**: focus `inputRefs.current[i+1]`
- **onPaste**: `try { const text = await navigator.clipboard.readText(); } catch { const text = e.clipboardData.getData('text'); }` — distribute chars: `text.split('').slice(0, length).forEach((char, idx) => { newValues[i + idx] = char; })` — focus last filled field

Render structure:
- `<div className={clsx(styles.pinInput, className)}>` — `flex gap-xs items-center`
- Map over `Array(length)`:
  - If `separator` and `index > 0 && index % separator === 0`: render `<span className={styles.separator}>-</span>` separator line
  - `<input key={index} ref={el => { inputRefs.current[index] = el; }} className={clsx(styles.field, values[index] && styles.filled)} value={values[index] || ''} onChange={...} onKeyDown={...} onPaste={handlePaste} type={type === 'password' ? 'password' : 'text'} inputMode="numeric" maxLength={1} placeholder={placeholder} disabled={disabled} autoComplete={otp ? 'one-time-code' : undefined} />`
- Password mode: apply `-webkit-text-security: disc` via CSS when `type="password"`

**PINInput.module.css**:
