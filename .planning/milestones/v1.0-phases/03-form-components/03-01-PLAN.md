**Checkbox.tsx** — Replace stub in `src/components/Checkbox/Checkbox.tsx`:
- Props interface `CheckboxProps`: `{ id?: string; name?: string; value?: boolean | null; defaultValue?: boolean | null; onChange?: (value: boolean | null) => void; rhs?: boolean; allowIndeterminate?: boolean; disabled?: boolean; children?: React.ReactNode; className?: string }`
- Use `useControllable<boolean | null>({ value, defaultValue: defaultValue ?? false, onChange })`
- Click handler cycle:
  - If `allowIndeterminate`: `true -> null -> false -> true`
  - If NOT `allowIndeterminate`: toggle `true <-> false`
- Render structure: `<div className={clsx(styles.checkbox, rhs && styles.rhs, disabled && styles.disabled, className)} onClick={handleClick}>`
  - Box: `<div className={clsx(styles.box, currentValue === true && styles.active, currentValue === null && styles.indeterminate)}>` with `transition-all duration-150 ease-out`
  - Box has `h-[1em] w-[1em] rounded-xs border border-primary-default` (default), `bg-primary-interaction-selected border-primary-interaction-selected` (active)
  - Check SVG (when value === true): `<svg viewBox="0 0 16 16" className="w-full h-full"><path d="M3.5 8L6.5 11L12.5 5" stroke="currentColor" strokeWidth="1.5" fill="none" strokeLinecap="round" strokeLinejoin="round"/></svg>`
  - Indeterminate SVG (when value === null): `<svg viewBox="0 0 16 16" className="w-full h-full"><line x1="4" y1="8" x2="12" y2="8" stroke="currentColor" strokeWidth="1.5" strokeLinecap="round"/></svg>`
  - SVG stroke is white (currentColor inherits from white text on active bg)
  - Label: if `children` provided AND (`id` or `name` exists), wrap in `<label htmlFor={id || name}>`; otherwise render `<span>` with `text-sm` styling
  - Gap between box and label: `gap-xs` (8px)
- If `rhs` is true, label renders on the left side (flex-row-reverse)
- Export as named: `export function Checkbox(...)`
