**RadioGroupContext** — Define inline in RadioGroup.tsx and export:
```typescript
export interface RadioGroupContextValue {
  selected: any;
  disabled: boolean;
  select: (value: any) => void;
}
export const RadioGroupContext = createContext<RadioGroupContextValue | null>(null);
```

**RadioGroup.tsx** — Replace stub:
- Props interface `RadioGroupProps`: `{ value?: any; defaultValue?: any; onChange?: (value: any) => void; vertical?: boolean; disabled?: boolean; options?: Array<any>; labelKey?: string; valueKey?: string; getObject?: boolean; children?: React.ReactNode; className?: string }`
- Use `useControllable<any>({ value, defaultValue, onChange })`
- Helper functions (port from Vue Group.vue):
  - `getLabel(opt)`: if option is object, return `opt[labelKey || 'label']`; else return `String(opt)`
  - `getValue(opt)`: if option is object, return `opt[valueKey || 'value']`; else return opt
  - `getDisabled(opt)`: if option is object, return `opt.disabled ?? false`; else return false
- Wrap children in `<RadioGroupContext.Provider value={{ selected: currentValue, disabled: disabled ?? false, select: setValue }}>`:
  - If `options` array provided, render `options.map(opt => <Radio key={getValue(opt)} groupValue={getValue(opt)} disabled={getDisabled(opt)}>{getLabel(opt)}</Radio>)`
  - If `children` provided (no options), render `{children}` directly
- Layout: `<div className={clsx(styles.radioGroup, vertical ? styles.vertical : styles.horizontal, className)}>`

**Radio.tsx** — Replace stub:
- Props interface `RadioProps`: `{ id?: string; name?: string; value?: boolean; defaultValue?: boolean; onChange?: (value: boolean) => void; groupValue?: any; disabled?: boolean; variant?: 'default' | 'onboarding'; children?: React.ReactNode; className?: string }`
- Read context: `const groupCtx = useContext(RadioGroupContext)`
- Determine selected state:
  - If `groupCtx` AND `groupValue !== undefined`: `isSelected = groupCtx.selected === groupValue`
  - Else: use `useControllable<boolean>({ value, defaultValue: defaultValue ?? false, onChange })` — `isSelected = currentValue`
- Click handler:
  - If `groupCtx && groupValue !== undefined`: `groupCtx.select(groupValue)`
  - Else: `setValue(true)` (Radio only goes to true, cannot deselect standalone — mirrors Vue behavior)
- Disabled: `groupCtx?.disabled || disabled`
- Render structure:
  - Outer: `<div className={clsx(styles.radio, styles[variant || 'default'], isSelected && styles.selected, isDisabled && styles.disabled, className)} onClick={handleClick}>`
  - Outer circle: `<div className={styles.outerCircle}>` — `w-[1em] h-[1em] rounded-full border-xxs` with `border-primary-default` (default), `border-current text-primary-interaction-selected` (selected)
  - Inner circle: `<div className={clsx(styles.innerCircle, isSelected && styles.innerCircleActive)}>` — `w-[.5em] h-[.5em] rounded-full bg-current transition-transform ease-out`, scale from 0 to 1 on selected
  - Label: render `children` with `text-xs` styling; wrap in `<label htmlFor={id || name}>` if id/name present
  - Gap: `gap-xs` (8px)
- Variant `onboarding`: outer circle is `text-xl` (larger), label is `text-sm`
