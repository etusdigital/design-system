Error handling with auto-dismiss:
```typescript
function showError(msg: string) {
  setHasError(true);
  setErrorMsg(msg);
  clearTimeout(errorTimerRef.current);
  errorTimerRef.current = setTimeout(() => {
    setHasError(false);
    setErrorMsg('');
  }, 2000);
}
// Cleanup on unmount:
useEffect(() => () => clearTimeout(errorTimerRef.current), []);
```

`addTag(rawValue: string)` function:
1. Trim the value
2. If `isNilAndBlank(trimmed)`: return (ignore empty)
3. If `mask`: apply `applyMask(trimmed, mask)` — if result different from trimmed, call `showError(ERROR_MESSAGES.INVALID_INPUT)` and return
4. If `!allowDuplicate` and tag exists in current tags: call `showError(ERROR_MESSAGES.DUPLICATE)` and return
5. If `max` and tags.length >= max: call `showError(ERROR_MESSAGES.MAX_VALUES)` and return
6. `setTags(prev => [...(prev ?? []), trimmed])`
7. `setNewTag('')`

`removeTag(index: number)` function:
- Only remove if `newTag` is empty (backspace removes last tag only when textarea is empty)
- `setTags(prev => (prev ?? []).filter((_, i) => i !== index))`

Key handlers on textarea:
- **Enter** (not Shift+Enter): `e.preventDefault()`, call `addTag(newTag)`
- **Tab**: `e.preventDefault()`, call `addTag(newTag)` (if newTag has content)
- **Backspace**: if `newTag === ''`, remove last tag via `removeTag(tags.length - 1)`
- **Paste**: intercept with `onPaste`, split by comma and newline: `text.split(/[,\n]/).map(s => s.trim()).filter(Boolean).forEach(addTag)`

Render structure:
1. **Label row** (if `labelValue`): `<div className={styles.labelRow}>` — Label + optional counter `{tags.length} / {max}`
2. **Container**: `<div className={clsx(styles.container, isFocused && styles.focused, (isError || hasError) && styles.error, disabled && styles.disabled, hasError && styles.shake)}>` — `border border-neutral-default rounded-base py-xs px-sm bg-white flex flex-wrap gap-xxs items-center`
   - Focus: `border-primary-interaction-default`
   - Error: `border-danger-default`
3. **Prepend icon** (if `icon` or PrependIcon child): `<Icon name={icon} />` or PrependIcon child
4. **Tags**: map over `tags` array — each tag rendered as:
   ```tsx
   <Tooltip text={String(tag)} key={index}>
     <StatusBadge text={String(tag)} color="neutral" closeable onClose={() => removeTag(index)} />
   </Tooltip>
   ```
5. **Textarea**: `<textarea ref={textareaRef} className={styles.textarea} rows={1} value={newTag} onChange={e => setNewTag(e.target.value)} onKeyDown={handleKeyDown} onPaste={handlePaste} onFocus={() => setIsFocused(true)} onBlur={() => setIsFocused(false)} placeholder={tags.length === 0 ? placeholder : ''} disabled={disabled} />`

File handling:

Computed values:
- `fileName`: if `multiple` and `Array.isArray(currentFile)` → `${currentFile.length} files selected`; else if single file → `currentFile.name`; else → ''
- `hasFile`: `currentFile !== null && (Array.isArray(currentFile) ? currentFile.length > 0 : true)`
- `getSvgSize(size)`: `{ xs: 56, sm: 66, md: 76, lg: 86, xl: 96 }[size || 'md']`
- Icon size map: `{ xs: 'text-4xl', sm: 'text-5xl', md: 'text-6xl', lg: 'text-7xl', xl: 'text-8xl' }[size || 'md']`
