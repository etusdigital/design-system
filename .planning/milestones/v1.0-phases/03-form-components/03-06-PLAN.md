**Slider.tsx** — Replace stub. This is a faithful port of the 589-line Vue Slider.vue. Read the Vue source thoroughly before implementing.

Props interface `SliderProps`:
```typescript
interface SliderProps {
  value?: number | [number, number];
  defaultValue?: number | [number, number];
  onChange?: (value: number | [number, number]) => void;
  size?: 'small' | 'medium' | 'large';
  max?: number;
  unit?: string;
  color?: string;
  showTooltip?: boolean;
  disabled?: boolean;
  vertical?: boolean;
  fillColors?: string[];
  isRange?: boolean;
  steps?: Array<{ label: string; value: number }>;
  neutralBackground?: boolean;
  className?: string;
}
```

Component: `export const Slider = React.forwardRef<HTMLDivElement, SliderProps>(function Slider(props, ref) { ... })`

Internal state and refs:
- `useControllable` for the value (number or [number, number])
- `const [isDragging, setIsDragging] = useState<boolean[]>([false, false])` — one flag per cursor (single=1, range=2)
- `const containerRef = useRef<HTMLDivElement>(null)` — merge with forwarded ref
- `const fillBarRef = useRef<HTMLDivElement>(null)`
- `const cursorRefs = useRef<(HTMLDivElement | null)[]>([])`

Core calculation function `calculateCursor()`:
- Read container dimensions: `containerRef.current.clientWidth` (horizontal) or `clientHeight` (vertical)
- For each cursor, compute position as percentage of max: `(cursorValue / max) * 100`
- Set cursor position via inline style: `left: ${percent}%` (horizontal) or `bottom: ${percent}%` (vertical)
- Set fill bar width/height: single mode fill from 0 to cursor; range mode fill between two cursors
- Use `getComputedStyle(el).getPropertyValue('--border-width-xs')` instead of `computedStyleMap()` for border offset calculation (jsdom compat)

Drag handlers:
- `startDragging(index)`: set `isDragging[index] = true` via new array
- `updateCursor(e: MouseEvent)`: calculate new value from mouse position relative to container bounds, clamp 0 to max, call setValue
- `updateCursorTouch(e: TouchEvent)`: same using `e.touches[0]`
- `stopDragging()`: set all isDragging to false

Window event listeners (in useEffect with empty deps):
```typescript
useEffect(() => {
  const handleMouseMove = (e: MouseEvent) => updateCursor(e);
  const handleMouseUp = () => stopDragging();
  const handleTouchMove = (e: TouchEvent) => updateCursorTouch(e);
  const handleTouchEnd = () => stopDragging();
  window.addEventListener('mousemove', handleMouseMove);
  window.addEventListener('mouseup', handleMouseUp);
  window.addEventListener('touchmove', handleTouchMove);
  window.addEventListener('touchend', handleTouchEnd);
  return () => {
    window.removeEventListener('mousemove', handleMouseMove);
    window.removeEventListener('mouseup', handleMouseUp);
    window.removeEventListener('touchmove', handleTouchMove);
    window.removeEventListener('touchend', handleTouchEnd);
  };
}, []);
```
IMPORTANT: Use refs (not state) for drag values inside handlers to avoid stale closures. Store updateCursor/stopDragging in refs that update on each render.

Initial calculation on mount — port Vue's `setTimeout`:
```typescript
useEffect(() => {
  const timer = setTimeout(() => calculateCursor(), 200);
  return () => clearTimeout(timer);
}, []);
```

Recalculate on value/size/vertical changes:
```typescript
useEffect(() => {
  const timer = setTimeout(() => calculateCursor(), 100);
  return () => clearTimeout(timer);
}, [currentValue, size, vertical]);
```

Render structure:
- Size class: `small` = `text-xs`, `medium` = `text-base`, `large` = `text-xl` (em-based sizing)
- Container: `<div ref={mergedRef} className={clsx(styles.slider, styles[size || 'medium'], vertical && styles.vertical, disabled && styles.disabled, className)}>`
- Track: `<div className={styles.track} onClick={handleTrackClick}>` — `h-[.5em] rounded-base` with `bg-primary-surface-highlight` (or `bg-neutral-surface-default` if `neutralBackground`)
- Fill bar: `<div ref={fillBarRef} className={styles.fillBar} style={fillBarStyle}>` — `bg-primary-interaction-default` (or custom `color` prop), absolute positioned within track
- Cursor(s): `<div ref={el => cursorRefs.current[i] = el} className={clsx(styles.cursor, isDragging[i] && styles.dragging)} style={cursorStyle} onMouseDown={() => startDragging(i)} onTouchStart={() => startDragging(i)}>` — `w-[1.25em] h-[1.25em] rounded-full border-2 border-primary-interaction-default bg-white cursor-grab`
  - If `showTooltip`: render `<div className={styles.tooltip}>{displayValue}{unit}</div>` above cursor
- Steps (if `steps` array): render step markers below track — `<div className={styles.stepsContainer}>` with each step as `<div className={styles.step} style={{left: (step.value/max)*100 + '%'}}>` containing a marker line and label text

Dynamic color via inline style:
- If `color` prop: cursor border color = `color`, fill bar bg = `color`
- If `fillColors` array: fill bar uses gradient or segmented colors (port Vue logic exactly)
- Apply via `style={{ borderColor: color }}` on cursor, `style={{ backgroundColor: color }}` on fill bar
