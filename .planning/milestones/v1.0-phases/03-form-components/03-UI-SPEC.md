# Phase 3 — UI Design Contract: Form Components

> Visual and interaction contract for 12 form components + Label utility. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (custom design system) |
| Preset | not applicable |
| Component library | none (custom components) |
| Icon library | Material Symbols (font-based `<span>`) |
| Font | Poppins (default), Inter (brius-theme) |

---

## Spacing Scale

Declared values from `--spacing-*` CSS custom properties (multiples of 4, except `2xxs`):

| Token | Value | Form Component Usage |
|-------|-------|----------------------|
| none | 0px | Zero gaps |
| 2xxs | 2px | Exception: micro adjustments only |
| xxs | 4px | Label-to-input gap, inner tag padding, checkbox/radio text gap, field-to-error-message gap |
| xs | 8px | Input vertical padding, icon gaps within inputs, PINInput field gaps, tag gaps |
| sm | 12px | Input horizontal padding, label spacing, RadioGroup/ToggleGroup item spacing |
| base | 16px | Toggle padding, Checkbox/Switch inline size (1em at 16px), number arrow spacing |
| lg | 20px | — |
| xl | 24px | Input min-height, PINInput field size (w/h), color display width |
| 2xl | 32px | Toggle secondary padding, slider cursor size |
| 3xl | 48px | PINInput field size (w/h 48px) |

Exceptions: `--spacing-2xxs: 2px` is not a multiple of 4; preserved from upstream design tokens for micro border offsets.

## Typography

Form components use 4 text roles with 2 weights:

| Role | Token | Size | Weight | Line Height | Usage |
|------|-------|------|--------|-------------|-------|
| Body (p3) | `text-sm` | 14px | `font-normal` (400) | `leading-lg` (120%) | Input/textarea content, checkbox labels, tag input text, error messages |
| Label (h5) | `text-sm` | 14px | `font-semibold` (600) | `leading-xs` (105%) | Field labels via Label component |
| Caption | `text-xs` | 12px | `font-normal` (400) | `leading-lg` (120%) | Radio labels, switch labels, counter text, hint messages |
| Counter | `text-sm` | 14px | `font-bold` (700) | default | Max-length counter (`3/50`), tag counter |

**Weights used in form components:** `normal` (400) and `semibold` (600), with `bold` (700) for counters only.

## Form-Specific Patterns

### Controlled/Uncontrolled Mode

Every form component supports dual modes via `useControllable` hook:

| Mode | Props Required | Behavior |
|------|---------------|----------|
| Controlled | `value` + `onChange` | Parent owns state; component reflects props |
| Uncontrolled | `defaultValue` (optional) | Component manages internal state via `useState` |

### Label Positioning

| Pattern | Components | Layout |
|---------|------------|--------|
| Top label with optional counter | Input, Textarea, TagInput | `flex justify-between items-center` above input |
| Inline label (right of control) | Checkbox, Radio, Switch | `inline-flex gap-xs items-center` |
| Inline label (content slot) | Toggle | `children` rendered inside button |
| No label | Slider, PINInput | Label managed externally by consumer |
| Top label (standalone) | FileUpload | `labelValue` prop rendered as upload CTA text |

### Helper/Error Text Placement

| Element | Position | Styling |
|---------|----------|---------|
| Error message | Below input container | `text-sm text-danger-foreground-low text-start` |
| Info message | Below FileUpload container | `text-sm text-neutral-foreground-low` |
| Hint message | Below TagInput container | `p3 text-neutral-foreground-low` (via `TagInput.Hint` slot) |
| Character counter | Top-right, beside label | `text-sm text-neutral-foreground-low font-bold` |

### Ref Forwarding

| Component | Ref Target | Imperative Methods |
|-----------|------------|-------------------|
| Input | Native `<input>` element | `.focus()`, `.select()`, `.blur()` |
| Textarea | Native `<textarea>` element | `.focus()`, `.select()`, `.blur()` |
| Slider | Container `<div>` element | none |
| PINInput | `useImperativeHandle` | `{ focus(), clear() }` |

### Compound Sub-Components

| Parent | Sub-Component | Replaces (Vue slot) |
|--------|--------------|---------------------|
| Input | `Input.PrependIcon` | `icon-slot` |
| Input | `Input.AppendIcon` | `appended-icon-slot` |
| TagInput | `TagInput.PrependIcon` | `icon-slot` |
| TagInput | `TagInput.AppendIcon` | `appended-icon-slot` |
| TagInput | `TagInput.Hint` | `hint-message` slot |
| FileUpload | `FileUpload.Preview` | `uploaded-file` slot |
