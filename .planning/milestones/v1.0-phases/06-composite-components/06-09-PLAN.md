Props interface `ColorPickerProps`:
- `value?: string` — color value as hex string (controlled)

**Internal state:**
- `useControllable<string>({ value, defaultValue: '#000000ff', onChange })` for the output color
- `useState<'hexa' | 'rgba' | 'hsla' | 'hsva' | 'hwb'>('hexa')` for active color type
- Internal HSVA representation: `useState<{h: number, s: number, v: number, a: number}>` — this is the canonical internal format
- Convert from model (hex) to HSVA on mount and when value changes externally

**Color area (saturation/value gradient):**
- Canvas element that draws a 2D gradient: horizontal = saturation (0-100), vertical = value (100-0 top to bottom)
- Background: linear gradient of hue color at full saturation from left (white) to right
- Overlay: linear gradient from top (transparent) to bottom (black)
- Cursor positioned at (s%, v%) of the area
- Drag interaction: mousedown on area sets cursor position, mousemove updates

**Hue slider:**
- Horizontal bar with rainbow gradient background (CSS conic-gradient or linear-gradient with stops at each 60deg hue)
- Cursor: circular, positioned at hue/360 * width
- Drag updates hue component

**Opacity slider (if showAlpha):**
- Horizontal bar with transparency gradient (checkerboard pattern + color)
- Cursor: circular, positioned at alpha * width
- Drag updates alpha component
