Port Calendar.vue (409 lines) + Day.vue (208 lines) + DateDialog.vue (56 lines) into a single Calendar.tsx with compound sub-components.

**Calendar props interface `CalendarProps`:**
- `value?: Date | Date[] | [Date[], Date[]]` — selected date(s), type depends on mode
- `defaultValue?: Date | Date[] | [Date[], Date[]]` — initial value
- `onChange?: (value: any) => void` — date selection change
- `type?: 'date' | 'period' | 'compare'` — selection mode, default `'date'`
- `minDate?: Date` — minimum selectable date
- `maxDate?: Date` — maximum selectable date
- `disabledDates?: Date[]` — specific dates to disable
- `lang?: string` — locale for month names, default `'en'`
- `className?: string`

**Define at module scope (3 components):**

1. **CalendarDay** (from Day.vue, 208 lines):
   - Props: `date: Date | null`, `selected: boolean`, `inRange: boolean`, `isRangeStart: boolean`, `isRangeEnd: boolean`, `isToday: boolean`, `isOutsideMonth: boolean`, `isDisabled: boolean`, `onClick: () => void`
   - Renders a single day cell in the month grid
   - Visual states per UI-SPEC:
     - Unselected: `text-sm text-neutral-foreground-high`, hover: `bg-neutral-surface-hover rounded-xs`
     - Selected (date mode): `bg-primary-surface-default text-primary-foreground-high rounded-xs border-xxs border-primary-default`
     - Range (period mode): `bg-primary-surface-hover` between start/end; endpoints: `bg-primary-interaction-default text-neutral-foreground-negative`
     - Today: `font-semibold`
     - Outside month: `text-neutral-foreground-disabled`
     - Disabled: `text-neutral-foreground-disabled cursor-not-allowed`
   - Null date = empty cell (padding)

2. **CalendarDateDialog** (from DateDialog.vue, 56 lines):
   - Props: `months: string[]`, `currentMonth: number`, `currentYear: number`, `onSelectMonth: (month: number) => void`, `onSelectYear: (year: number) => void`
   - Renders month/year picker grid (same card styling as SelectContent)
   - Month buttons in 3x4 grid, year range buttons
   - Active month/year highlighted

3. **Calendar** (from Calendar.vue, 409 lines):
   - State: `currentMonth` and `currentYear` (useState, initialized from value or today)
   - `isBackRef = useRef(false)` — controls slide direction (NOT useState per RESEARCH.md pitfall 4)
   - `const [showDateDialog, setShowDateDialog] = useState(false)` for month/year picker
   - Month grid: `getArrayMonthDay(new Date(currentYear, currentMonth, 1))` from utils
   - Month names: `getMonths(lang)` from utils
   - Navigation: prev/next month buttons with Material Symbols `chevron_left`/`chevron_right`
     - `aria-label="Previous month"` and `aria-label="Next month"`
   - Prev month: set `isBackRef.current = true`, then update month/year
   - Next month: set `isBackRef.current = false`, then update month/year
   - **Slide-fade transition**: `useTransition` on a `transitionKey` that changes each month navigation. CSS class: `isBackRef.current ? 'slide-fade-back' : 'slide-fade-forward'`
   - Date mode: click day sets value to that Date
   - Period mode: first click = range start, second click = range end. If start > end, swap.
   - Compare mode: two range arrays `[Date[], Date[]]`
   - Day of week headers: short names (Sun, Mon, ...) from locale
   - `capitalizeFirstLetter` from utils for month name display
