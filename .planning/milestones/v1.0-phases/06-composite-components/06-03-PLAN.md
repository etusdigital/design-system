Port Select.vue (310 lines) to React TSX. Select is the core dropdown using SelectContainer infrastructure.

Props interface `SelectProps`:
- `value?: any` тАФ selected value (controlled). Single: primitive or object. Multi: array.
- `defaultValue?: any` тАФ initial value (uncontrolled)
- `onChange?: (value: any) => void` тАФ selection change callback
- `options: any[]` тАФ list of selectable options
- `labelKey?: string` тАФ key for display label in object options, default `'label'`
- `valueKey?: string` тАФ key for value in object options, default `'value'`
- `multiple?: boolean` тАФ enable multi-select with checkboxes, default `false`
- `searchable?: boolean` тАФ enable search input in dropdown, default `false`
- `clearable?: boolean` тАФ show clear button, default `false`
- `disabled?: boolean` тАФ default `false`
- `labelValue?: string` тАФ trigger label text, default `''`
- `getObject?: boolean` тАФ emit full option object (not just valueKey), default `false`
- `isError?: boolean` тАФ error border, default `false`
- `errorMessage?: string` тАФ default `''`
- `infoMessage?: string` тАФ default `''`
- `required?: boolean` тАФ default `false`
- `placeholder?: string` тАФ search input placeholder
- `children?: React.ReactNode` тАФ custom option renderer (compound sub-component slot)
- `className?: string`

Implementation:
- `useControllable({ value, defaultValue: multiple ? [] : null, onChange })` for selected value
- `useState('')` for search text (internal)
- `useState(false)` for expanded state passed to SelectContainer
- `getLabel(option)`: `isObject(option) ? option[labelKey] : String(option)`
- `getValue(option)`: `isObject(option) ? option[valueKey] : option`
- `isSelected(option)`: for single mode: `getValue(model) === getValue(option)`. For multi: `model.some(m => getValue(m) === getValue(option))`
- `selectOption(option)`: for single: set model to `getObject ? option : getValue(option)`, collapse dropdown. For multi: toggle option in/out of array.
- Filter logic: `filteredOptions = searchText ? options.filter(o => getLabel(o).toLowerCase().includes(searchText.toLowerCase())) : options`
- Display label: for single, `getLabel(selectedOption)`. For multi, show count or comma-separated.
- Pass to SelectContainer: `options` prop as mapped `<Option>` elements. Each Option: `selected={isSelected(option)}`, `onClick={() => selectOption(option)}`. If `multiple`, render `<Checkbox value={isSelected(option)} className="pointer-events-none" />` before label text.
- Pass to SelectContent: `searchable`, `value={searchText}`, `onChange={setSearchText}`.
- If `clearable` and has value, pass `actions` prop with clear Button.
- Use `isObject` from `src/utils/index.ts`.
