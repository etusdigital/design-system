**Tab.tsx** — Port from Tab.vue (101 lines). Simple tab navigation component.

Props interface `TabProps`:
- `value?: number` — active tab index (controlled)
- `defaultValue?: number` — initial active tab (uncontrolled)
- `onChange?: (value: number) => void` — tab change callback
- `options: Array<string | { label: string; [key: string]: any }>` — tab items
- `labelKey?: string` — key for label in object options, default `'label'`
- `notCard?: boolean` — remove card background, default `false`
- `children?: React.ReactNode`
- `className?: string`

Implementation:
- `useControllable<number>({ value, defaultValue: 0, onChange })` — initialize to 0 (first tab) when no value
- `getLabel(option)`: if string, return it; if object, return `option[labelKey]`
- Render tab buttons in a flex row. Each button: `onClick={() => setModel(index)}`
- Active tab: apply active CSS class (shadow + bg change)
- If `notCard` is false, container has background highlight

**Pagination.tsx** — Port from Pagination.vue (138 lines).

Props interface `PaginationProps`:
- `value?: number` — current page (1-indexed, controlled)
- `defaultValue?: number` — initial page (uncontrolled)
- `onChange?: (value: number) => void` — page change callback
- `length: number` — total number of pages
- `visiblePages?: number` — max visible page buttons, default `5`
- `className?: string`

- Prev button: Material Symbols `chevron_left`, disabled when `model === 1`
- Next button: Material Symbols `chevron_right`, disabled when `model === length`
- Ellipsis items: render `...` text, not clickable
- Active page: primary background, white text

Port Stepper.vue (359 lines) but ONLY version 2 — drop version 1 (arrow breadcrumbs) entirely. Remove `version` prop.

Props interface `StepperProps`:
- `value?: number` — active step index (controlled)
- `defaultValue?: number` — initial step (uncontrolled)
- `onChange?: (value: number) => void` — step change callback
- `options: Array<{ [key: string]: any }>` — step items
- `labelKey?: string` — key for label text, default `'label'`
- `iconKey?: string` — key for icon name, default `'icon'`
- `background?: string` — dynamic background color for connector blending
- `size?: 'medium' | 'large'` — circle size, default `'medium'`
- `noClick?: boolean` — disable step click navigation, default `false`
- `className?: string`

Implementation:
- `useControllable<number>({ value, defaultValue: 0, onChange })`
- Track `passedIn` as a `Set<number>` in state — steps that have been visited
- Track `biggerStepSelected` as boolean state — true if user clicked a step ahead of current
- When `model` changes via useEffect: add previous step to passedIn set, set biggerStepSelected if new > old
- CSS variable injection: `style={{ '--stepper-bg': background } as React.CSSProperties}` on root div
- Step states: `active` (current), `past` (in passedIn set or index < model and not skipped), `skipped` (in passedIn AND biggerStepSelected jumped over it), `default` (future)
- Circle sizes: medium = 32px (`w-[32px] h-[32px]`), large = 40px (`w-[40px] h-[40px]`)
- Connector line between steps: `flex-1 h-[2px]`, colored by state
- Each step renders: circle (with Icon inside) + label text below + connector line between steps
- Step click: `onClick={() => { if (!noClick) setModel(index) }}`
- Use `isObject` from `src/utils/index.ts` for option type checking
