- If `isMobile`, force `position = 'bottom'` and `width = '100%'`
- Root div: `className={clsx('drawer', position, isActive && 'active', className)}`
- `style={{ width, height, maxWidth: 'calc(100% - var(--spacing-xl))', maxHeight: 'calc(100% - var(--spacing-xl))' }}`

Port Accordion.vue (135 lines) to React TSX. Accordion uses ResizeObserver + MutationObserver for content height, matching the SelectContainer observer pattern.

Props interface `AccordionProps`:
- `value?: boolean` — expanded state (controlled)
- `defaultValue?: boolean` — initial expanded (uncontrolled)
- `onChange?: (value: boolean) => void` — expanded change callback
- `duration?: number` — transition duration in ms, default `300`, clamped to 150–1000
- `noShadow?: boolean` — remove card shadow, default `false`
- `disabled?: boolean` — prevent toggle, default `false`
- `header?: React.ReactNode` — header content (replaces Vue default slot header area)
- `children?: React.ReactNode` — collapsible body content
- `className?: string`

Implementation:
- `useControllable<boolean>({ value, defaultValue: false, onChange })` for expanded
- `const clampedDuration = Math.min(1000, Math.max(150, duration ?? 300))`
- `cardRef = useRef<HTMLDivElement>(null)`, `contentRef = useRef<HTMLDivElement>(null)`
- `useLayoutEffect`: when `!isExpanded`, set `contentRef.current.style.maxHeight = '0px'` to prevent initial flash
- `useEffect([], [])`: register ResizeObserver on cardRef + contentRef, MutationObserver on contentRef (childList, subtree, attributes with attributeFilter `['data-max-height']`). Both call `resize()`. Cleanup on unmount.
- `useEffect` (no deps — runs every render): call `resize()` to handle Vue's `onUpdated` equivalent
- `resize()` function: reads `contentRef.current.scrollHeight`, sets `maxHeight` to `${scrollHeight}px` when expanded, `'0px'` when collapsed
- Toggle: `onClick` on header row calls `setExpanded(!isExpanded)` if not disabled
- Chevron icon: Material Symbols `expand_more`, rotates 180deg when expanded via CSS class
