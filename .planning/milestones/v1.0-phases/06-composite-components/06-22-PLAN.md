**Restructure CalendarDateDialog:**
Currently it renders month and year grids simultaneously. The Vue version has:
1. A header row with "month" and "year" as clickable text buttons
2. Clicking "month" toggles the month grid panel
3. Clicking "year" toggles the year list panel
4. Only ONE panel shows at a time (month OR year, not both)
5. The whole dialog is wrapped in a Card component

Add state for active panel:
```typescript
const [activePanel, setActivePanel] = useState<'month' | 'year' | null>(null);
```

Restructure the render:
```tsx
function CalendarDateDialog({ ... }) {
  const [activePanel, setActivePanel] = useState<'month' | 'year' | null>(null);

  return (
    <Card className={clsx(styles.dateDialog, isOpen && styles.dialogEnter, !isOpen && styles.dialogLeave)} noShadow={false}>
      {/* Header with month/year toggle buttons */}
      <div className={styles.dialogHeader}>
        <button
          className={clsx(styles.headerToggle, activePanel === 'month' && styles.headerToggleActive)}
          onClick={() => setActivePanel(prev => prev === 'month' ? null : 'month')}
        >
          {monthName}
        </button>
        <button
          className={clsx(styles.headerToggle, activePanel === 'year' && styles.headerToggleActive)}
          onClick={() => setActivePanel(prev => prev === 'year' ? null : 'year')}
        >
          {year}
        </button>
      </div>

      {/* Month grid panel */}
      {activePanel === 'month' && (
        <div className={styles.monthGrid}>
          {months.map((month, i) => (
            <button
              key={i}
              className={clsx(styles.monthButton, currentMonth === i && styles.monthActive)}
              onClick={() => { handleMonthSelect(i); setActivePanel(null); }}
            >
              {month}
            </button>
          ))}
        </div>
      )}

      {/* Year list panel */}
      {activePanel === 'year' && (
        <div className={styles.yearList}>
          {yearRange.map(yr => (
            <button
              key={yr}
              className={clsx(styles.yearButton, currentYear === yr && styles.yearActive)}
              onClick={() => { handleYearSelect(yr); setActivePanel(null); }}
            >
              {yr}
            </button>
          ))}
        </div>
      )}
    </Card>
  );
}
```

**Year range:** Generate an array of years around the current year, e.g., `[currentYear - 10, ..., currentYear + 10]`. Use `useEffect` to scroll the active year into view when the year panel opens.

**Animation:** Use the `.dialogEnter` / `.dialogLeave` CSS classes (added in Task 1) to animate the Card appearing/disappearing. Use the existing `useTransition` hook from the codebase if available, or manage via a `isVisible` state with setTimeout for the leave animation.

**Ensure CalendarDateDialog is toggled from the main Calendar component** via the month/year header text. The main calendar header should show "March 2026" as clickable text that opens CalendarDateDialog.
