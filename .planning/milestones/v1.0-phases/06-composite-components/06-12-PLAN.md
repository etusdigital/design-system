Conversion rules (Vue → React):
- `@storybook/vue3` → `@storybook/react`
- `import Foo from './Foo.vue'` → `import { Foo } from './Foo'`
- `v-model="args.x"` → `value={x} onChange={setX}` (useState inside render function)
- `v-model:expanded="args.x"` → `expanded={exp} onExpandedChange={setExp}` (useState)
- `:prop="val"` → `prop={val}`
- Vue slot `<template #option="{ option, index }">...</template>` → React render prop `renderOption={(option, index) => ...}` (check actual component for the exact prop name)
- Vue slot `<template #search-label>text</template>` → React prop `searchLabel="text"` or `searchLabel={<>text</>}` (check actual component)
