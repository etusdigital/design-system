**Input.tsx** — Replace stub with full implementation wrapping `React.forwardRef`:

Props interface `InputProps`:
```typescript
interface InputProps {
  value?: string;
  defaultValue?: string;
  onChange?: (value: string) => void;
  labelValue?: string;
  type?: 'text' | 'number' | 'password' | 'search' | 'file' | 'email' | 'url' | 'domain' | 'color';
  mask?: string;
  max?: number;
  min?: number;
  step?: number;
  errorMessage?: string;
  infoMessage?: string;
  disabled?: boolean;
  isError?: boolean;
  required?: boolean;
  placeholder?: string;
  textAlign?: 'left' | 'center' | 'right';
  tooltipMinWidth?: number;
  icon?: string;          // Material Symbols icon name for prepend
  appendIcon?: string;    // Material Symbols icon name for append
  onFocus?: () => void;
  onBlur?: () => void;
  children?: React.ReactNode;  // for compound sub-components
  className?: string;
}
```

Component: `export const Input = React.forwardRef<HTMLInputElement, InputProps>(function Input(props, ref) { ... })`

Internal state:
- `useControllable<string>({ value, defaultValue: defaultValue ?? '', onChange })` for the text value
- `const [isFocused, setIsFocused] = useState(false)`
- `const [hasError, setHasError] = useState(false)`
- `const [showPassword, setShowPassword] = useState(false)` (for type="password")
- `const [fileName, setFileName] = useState('')` (for type="file")
- `const internalRef = useRef<HTMLInputElement>(null)` — merge with forwarded ref via callback ref pattern

Compound sub-components (defined as static properties on Input):
```typescript
function PrependIcon({ children }: { children: React.ReactNode }) { return <>{children}</>; }
function AppendIcon({ children }: { children: React.ReactNode }) { return <>{children}</>; }
Input.PrependIcon = PrependIcon;
Input.AppendIcon = AppendIcon;
```

Extract PrependIcon and AppendIcon children using `React.Children.forEach` to find matching sub-component types, similar to Phase 2 pattern.

Render structure (top-to-bottom):
1. **Label row** (if `labelValue`): `<div className={styles.labelRow}>` — flex row with justify-between
   - Left: `<Label labelValue={labelValue} infoMessage={infoMessage} tooltipMinWidth={tooltipMinWidth} required={required} />`
   - Right (if `type === 'text' && max`): `<span className={styles.counter}>{(currentValue ?? '').length}/{max}</span>` — `text-sm font-bold text-neutral-foreground-low`

2. **Input container**: `<div className={clsx(styles.inputContainer, isFocused && styles.focused, (isError || hasError) && styles.error, disabled && styles.disabled)}>` — `flex items-center gap-xs py-xs px-sm min-h-xl rounded-sm outline outline-xxs outline-neutral-default bg-emphasis transition-[outline-color] duration-100 delay-100`
   - Focus: `outline-primary-default`
   - Error: `outline-danger-default`
   - Disabled: `outline-neutral-disabled bg-neutral-surface-disabled pointer-events-none`

3. **Prepend icon area**:
   - If `type === 'search'`: render `<Icon name="search" className="text-neutral-foreground-low" />`
   - Else if `icon` prop: render `<Icon name={icon} />`
   - Else if PrependIcon compound child found: render it

4. **Native input**: `<input ref={mergedRef} className={clsx(styles.inputContent, textAlign && styles[textAlign])} value={...} onChange={handleChange} onFocus={handleFocus} onBlur={handleBlur} type={resolvedType} placeholder={placeholder} disabled={disabled} />`
   - `resolvedType`: for `password` use `showPassword ? 'text' : 'password'`; for `number` use `'text'` (custom arrows, hide native spinners); for `search`/`email`/`url`/`domain` use `'text'`; for `file` use `'file'` with hidden input
   - For `type='file'`: do NOT use useControllable; use `onChange` from native input `e.target.files`; display `fileName` state
   - `styles.inputContent`: `@apply flex-1 bg-transparent outline-none text-sm text-neutral-foreground-high placeholder:text-neutral-foreground-low;`

5. **Append icon area**:
   - If `type === 'password'`: render toggle icon `<Icon name={showPassword ? 'visibility_off' : 'visibility'} className="cursor-pointer" onClick={() => setShowPassword(!showPassword)} />`
   - If `type === 'number'`: render up/down arrows column — `<div className={styles.numberArrows}>` with `<Icon name="arrow_drop_up" onClick={increment} className="text-2xl h-[.7em] cursor-pointer" />` and `<Icon name="arrow_drop_down" onClick={decrement} className="text-2xl h-[.7em] cursor-pointer" />`
   - Else if `appendIcon` prop: render `<Icon name={appendIcon} />`
   - Else if AppendIcon compound child found: render it

6. **Error message** (if `errorMessage` and `(isError || hasError)`): `<p className={styles.errorMessage}>{errorMessage}</p>` — `text-sm text-danger-foreground-low text-start`

Input change handler (`handleChange`):
- If `mask`: apply `applyMask(e.target.value, mask)` before setValue
- If `type === 'email'`: validate with `isValidEmail()` — set `hasError` if invalid on blur
- If `type === 'domain'`: validate with `isValidDomain()`
- If `type === 'url'`: validate with `isValidUrl()`
- If `max && type === 'text'`: enforce max length — `e.target.value.slice(0, max)`
- If `type === 'number'`: validate numeric input, clamp between min/max

Number increment/decrement:
- `increment()`: `setValue(String(Math.min(Number(currentValue || 0) + (step || 1), max ?? Infinity)))`
- `decrement()`: `setValue(String(Math.max(Number(currentValue || 0) - (step || 1), min ?? -Infinity)))`

`type="color"` handling: Add TODO comment at the top of the type switch:
```typescript
// TODO: type="color" deferred to Phase 6 — requires ColorPicker + FloatCard integration
```
For now, render a standard text input when type="color" (no color picker UI).

Ref merging: Use callback ref to assign both forwarded ref and internal ref:
```typescript
const mergedRef = (el: HTMLInputElement | null) => {
  internalRef.current = el;
  if (typeof ref === 'function') ref(el);
  else if (ref) (ref as React.MutableRefObject<HTMLInputElement | null>).current = el;
};
```
