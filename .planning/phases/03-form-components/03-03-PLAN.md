**ToggleGroupContext** — Define inline in ToggleGroup.tsx and export:
```typescript
export interface ToggleGroupContextValue {
  selected: any;
  disabled: boolean;
  type: 'default' | 'secondary';
  select: (value: any) => void;
}
export const ToggleGroupContext = createContext<ToggleGroupContextValue | null>(null);
```

**ToggleGroup.tsx** — Replace stub:
- Props interface `ToggleGroupProps`: `{ value?: any; defaultValue?: any; onChange?: (value: any) => void; vertical?: boolean; disabled?: boolean; type?: 'default' | 'secondary'; options?: Array<any>; labelKey?: string; valueKey?: string; getObject?: boolean; children?: React.ReactNode; className?: string }`
- Use `useControllable<any>({ value, defaultValue, onChange })`
- Same `getLabel`/`getValue`/`getDisabled` helpers as RadioGroup (port from Group.vue)
- Provider: `<ToggleGroupContext.Provider value={{ selected: currentValue, disabled: disabled ?? false, type: type ?? 'default', select: setValue }}>`
- If `options`, map to Toggle children: `<Toggle key={getValue(opt)} groupValue={getValue(opt)} disabled={getDisabled(opt)}>{getLabel(opt)}</Toggle>`
- Layout depends on `type`:
  - `default` + horizontal: connected pill row — no right border between items, first gets `rounded-l-sm`, last gets `rounded-r-sm` (via CSS `.default [&>*]:border-r-0 [&>*:last-child]:border-r`)
  - `default` + vertical: connected pill column — no bottom border, first `rounded-t-sm`, last `rounded-b-sm`
  - `secondary` + horizontal: `gap-xs` (8px) between items, each Toggle independently rounded
  - `secondary` + vertical: `gap-xs` vertical, each independently rounded

**ToggleGroup.module.css**:
- `.toggleGroup` — `@apply flex;`
- `.vertical` — `@apply flex-col;`
- `.default` — connected pill styles with border collapse between children
- `.secondary` — `@apply gap-xs;` (flex-wrap if needed)
- `.secondaryVertical` — `@apply gap-xs flex-col;`

**Toggle.tsx** — Replace stub:
- Props interface `ToggleProps`: `{ value?: boolean; defaultValue?: boolean; onChange?: (value: boolean) => void; id?: string; name?: string; groupValue?: any; disabled?: boolean; type?: 'default' | 'secondary'; children?: React.ReactNode; className?: string }`
- Read context: `const groupCtx = useContext(ToggleGroupContext)`
- Determine active state:
  - If `groupCtx && groupValue !== undefined`: `isActive = groupCtx.selected === groupValue`
  - Else: use `useControllable<boolean>({ value, defaultValue: defaultValue ?? false, onChange })` — `isActive = currentValue`
- Resolve type: `resolvedType = groupCtx?.type ?? type ?? 'default'`
- Click handler:
  - If `groupCtx && groupValue !== undefined`: `groupCtx.select(groupValue)`
  - Else: `setValue(true)` (Toggle is always-on once activated, mirrors Vue behavior)
- Disabled: `groupCtx?.disabled || disabled`
- Render as `<button>` element:
  - `className={clsx(styles.toggle, styles[resolvedType], isActive && styles.active, isDisabled && styles.disabled, className)}`
  - Content: `{children}` rendered inside button
- Styling:
  - Default type: `min-h-3xl min-w-9xl py-xs px-base border-xxs border-current text-xs font-bold uppercase tracking-wider`
    - Active: `bg-primary-interaction-default text-neutral-foreground-negative` (green bg, white text)
    - Inactive: `bg-transparent text-current`
  - Secondary type: `py-base px-2xl rounded-base border-xxs border-current text-xs font-bold uppercase tracking-wider`
    - Active: `bg-primary-surface-highlight text-primary-interaction-default` (light green bg, green text)
    - Inactive: `bg-transparent text-current`

**Toggle.module.css**:
- `.toggle` — `@apply flex items-center justify-center cursor-pointer select-none text-xs font-bold uppercase tracking-wider border border-current transition-all duration-150 ease-out;`

- Secondary active handled via compound selector `.secondary.active`
