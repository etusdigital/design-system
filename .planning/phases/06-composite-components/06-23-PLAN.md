3. Implement the Vue layout structure:
   ```
   [Logo circle] | [Dropdown nav] ............. [Notification bell] [Avatar]
   ```

   Left side:
   - Default SVG logo (a circle with company initials) OR custom `logo` prop
   - Vertical divider (1px border-right, full height)
   - Single `<Dropdown>` component with `options` prop for navigation

   Right side (flex with gap):
   - Notification bell: `<Icon name="notifications" />` wrapped in `<FloatCard>` that shows `notifications` content on click
   - Avatar: `<Avatar>` component or custom `avatar` prop

   ```tsx
   <nav className={clsx(styles.navbar, className)}>
     <div className={styles.left}>
       <div className={styles.logo}>
         {logo || (
           <svg width="32" height="32" viewBox="0 0 32 32">
             <circle cx="16" cy="16" r="16" fill="var(--primary-interaction-default)" />
           </svg>
         )}
       </div>
       <div className={styles.divider} />
       <Dropdown
         options={options}
         value={value}
         onChange={onChange}
         hideArrow
       />
     </div>
     <div className={styles.right}>
       {showNotifications !== false && (
         <FloatCard content={notifications}>
           <button className={styles.notificationButton}>
             <Icon name="notifications" />
           </button>
         </FloatCard>
       )}
       <div className={styles.avatarWrapper}>
         {avatar || <Avatar size="sm" />}
       </div>
     </div>
   </nav>
   ```
