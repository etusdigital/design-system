- `v-model` → `value + onChange` with useState
- `v-model:expanded` → `expanded + onExpandedChange` with useState

IMPORTANT — Dropdown compound component pattern (per STATE.md):
  The React Dropdown uses static sub-components: Dropdown.Options and Dropdown.Option
  Import: `import { Dropdown } from './Dropdown'`
  The `children` trigger slot becomes the `children` prop of Dropdown (for custom trigger).
  The options array prop is used for the auto-rendered options list.

  children (custom trigger slot)
