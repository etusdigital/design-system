**Top-level Dropdown props interface `DropdownProps`:**
- `value?: any` ŌĆö selected value (controlled)
- `defaultValue?: any` ŌĆö initial value (uncontrolled)
- `onChange?: (value: any) => void` ŌĆö selection change
- `options: any[]` ŌĆö nested options (each may have `options` array for sub-groups)
- `labelKey?: string` ŌĆö default `'label'`
- `valueKey?: string` ŌĆö default `'value'`
- `getObject?: boolean` ŌĆö emit full object, default `false`
- `disabled?: boolean` ŌĆö default `false`
- `labelValue?: string` ŌĆö trigger label, default `''`
- `searchable?: boolean` ŌĆö enable search in dropdown, default `false`
- `alignRight?: boolean` ŌĆö align dropdown right, default `false`
- `isError?: boolean` ŌĆö default `false`
- `errorMessage?: string` ŌĆö default `''`
- `children?: React.ReactNode` ŌĆö custom content override
- `className?: string`

**Implementation:**

Define three module-scope named functions (per RESEARCH.md pitfall 5 ŌĆö avoid inline component definitions):

1. `function DropdownOption({ option, ... })` ŌĆö renders a single option item
   - Props: `option`, `labelKey`, `valueKey`, `selected`, `onClick`, `depth`
   - If option has nested `options` array: render group header + recursive `DropdownOptions`
   - Group headers: `text-xs font-semibold text-neutral-foreground-low`
   - Option items: `text-sm font-normal`, hover `bg-neutral-surface-hover`, selected `text-primary-interaction-selected`
   - Click handler: call parent's selectOption

2. `function DropdownOptions({ options, ... })` ŌĆö renders a list of options, handles recursion
   - Maps over options, renders `DropdownOption` for each
   - Passes depth for indentation of nested groups

3. `function Dropdown(props)` ŌĆö main component
   - `useControllable({ value, defaultValue: null, onChange })` for selected value
   - `useState('')` for search text (if searchable)
   - `useState(false)` for expanded
   - `selectOption(option)`: set model to `getObject ? option : getValue(option)`, collapse dropdown
   - Display label: `getLabel(selectedOption)` or `labelValue`
   - Wraps in `<ExpandableContainer>`, passes `card` prop containing the options list
   - If searchable, render Input inside card for filtering
   - Filter logic: flatten nested options, filter by search text on label
   - Use `isObject` from `src/utils/index.ts`

4. Attach as static properties:
   ```typescript
   Dropdown.Options = DropdownOptions;
   Dropdown.Option = DropdownOption;
   ```
