  <action>
    For EACH of the 20 MDX files listed above, read the file and apply these transforms:

    **Transform rules (apply in order):**

    1. **Code fence language**: Replace all ` ```vue ` with ` ```tsx `

    2. **Remove template wrappers**: Delete `<template>` and `</template>` lines entirely. The JSX content inside becomes the code example directly.

    3. **Remove script blocks**: Delete entire `<script setup>` ... `</script>` blocks and any `defineProps`, `defineEmits`, `ref()`, `computed()` lines within them. Replace with React equivalent if the example needs state:
       ```tsx
       const [value, setValue] = useState(false);
       ```

    4. **Event syntax**: Replace `@eventName="handler"` with `onEventName={handler}`:
       - `@click="handler"` -> `onClick={handler}`
       - `@change="handler"` -> `onChange={handler}`
       - `@input="handler"` -> `onInput={handler}`
       - `@update:modelValue="handler"` -> `onChange={handler}`

    5. **v-bind shorthand**: Replace `:prop="value"` with `prop={value}`:
       - `:disabled="true"` -> `disabled={true}` (or just `disabled`)
       - `:progress="0.3"` -> `progress={0.3}`
       - `:options="items"` -> `options={items}`

    6. **v-model**: Replace `v-model="val"` with `value={val} onChange={setVal}` or just show the prop without state management since these are documentation examples.

    7. **v-if / v-show**: Replace `v-if="condition"` with `{condition && <Component />}` or note in a comment.

    8. **v-for**: Replace `v-for="item in items"` with `.map()` pattern:
       ```tsx
       {items.map(item => <Component key={item.id} />)}
       ```

    9. **Slot syntax**: Replace `#slotName` or `v-slot:slotName` with React children or named props. Default slot content just becomes children.

    10. **Import statement at top**: Ensure the import references `.stories.tsx` (not `.stories.ts` or `.stories.vue`):
        ```
        import * as ComponentName from './ComponentName.stories.tsx';
        ```
        Most should already be correct since stories were migrated to TSX in prior phases.

    11. **Vue-specific text**: Replace "Vue component" with "React component" in prose. Replace references to "slots" with "children/props". Replace "emit" with "callback prop".

    **IMPORTANT**: Keep all Storybook `<Canvas>`, `<Meta>`, `<Controls>` blocks unchanged — they are framework-agnostic MDX.

    **IMPORTANT**: For simple examples that just show component usage with static props, the JSX is nearly identical to the Vue template minus the `<template>` wrapper. Don't over-complicate simple examples.

    **Example transformation** (Button.mdx):
    Before:
    ```vue
    <template>
        <Button @click="handleClick">Click me</Button>
    </template>
    ```
    After:
    ```tsx
    <Button onClick={handleClick}>Click me</Button>
    ```

    Process all 20 files. Read each file, apply transforms, write the updated file.
