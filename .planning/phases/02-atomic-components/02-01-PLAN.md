3. **Create `Component.module.css`** (and `colors.module.css` if color variants exist):
   - Extract CSS from Vue `<style scoped>` block.
   - Convert to CSS Module format.
   - Preserve `@apply` Tailwind directives as-is.
   - Apply module classes directly to each element (no CSS nesting for child components).
4. **Create `Component.stories.tsx`**:
   - Read existing `.stories.ts` (Vue) and adapt for React.
   - Change import from `@storybook/vue3` to `@storybook/react`.

   - Keep same CSF structure (args, argTypes, story names).
5. **Delete Vue files**: Delete `.vue` and `.stories.ts` files after React versions are verified.
